Rule
Tracking tightens with size. At 160 px+, −.04em closes the gaps between wide Spartan letterforms without jamming the M into the o. At nav scale, back off to −.035em to preserve legibility. Below 14 px, use −.025em or 0 — don't crush the type.

Weight study — why 700, not 900

Running every available weight at the same 72 px so the difference is legible, not theoretical.

Why 700
900 (Black) is what PrizePicks / Overtime / hype-culture brands use — loud and square. 700 (Bold) keeps presence without volume. League Spartan's wide geometric forms already carry confidence at 700; going heavier makes the counters close up and the word start to look like a brick. Bold = calibrated presence, which fits "coaches the coach" better than a shout.

import { League_Spartan } from "next/font/google";

const spartan = League_Spartan({
  subsets: ["latin"],
  weight: ["400", "500", "700", "900"],
  variable: "--font-display",
  display: "swap",
});

/* 2. Attach to <html> via className={spartan.variable} */

/* 3. Wordmark component — src/components/brand/Wordmark.tsx */
export function Wordmark({ size = "nav" }) {
  const styles = {
    nav:    { fontSize: "22px", letterSpacing: "-0.035em" },
    h1:     { fontSize: "48px", letterSpacing: "-0.04em"  },
    hero:   { fontSize: "clamp(96px, 12vw, 168px)", letterSpacing: "-0.04em" },
    footer: { fontSize: "14px", letterSpacing: "-0.025em" },
  }[size];
  return (
    <span
      className="font-display font-bold"
      style={{ ...styles, lineHeight: ".88" }}
    >
      Motion
    </span>
  );
}
