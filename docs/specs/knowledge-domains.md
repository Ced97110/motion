### Defense Simulator Details

Three interactive modules from Herb Brown's championship defense book:

**Help Defense Rotations** — weak-side nail/elbow/box positioning. User drags 5 defenders. AI grades: "X3 is too high — drop to the nail position."

**Pick & Roll Defense** — 5 coverage options (hard hedge, soft show, switch, drop, blitz). Run the play, see if coverage holds. Edge cases: "What if the guard goes under?" "What if the big pops?"

**Low Post Trap** — double-team and rotation responsibilities. When/where to trap, stealing passing lanes.

All feedback cites the source book with chapter and page.

### Live Stat Tracking

Not a standalone feature — the data collection mechanism that feeds the intent engine and builds the proprietary dataset.

**How**: Coach taps stats during game. One tap per stat. Points, rebounds, assists, steals, blocks, turnovers, FG/3PT/FT. Shot chart (tap court for location).

**Immediate value**: Real-time box score. Halftime AI insights. Post-game report.

**Export**: CSV, MaxPreps format, PDF. Coach owns their data.
